This is a lazy-load javascript file,it will work with you better.
Usage:
First step,you should add the class named "lazy" to your images that will use lazy load.
Then set the default src to its src and set the real src to the attribute named "data-src".
Finally,invoke the function myScrollListener for window.onscroll and invoke firstLoad for window.onload.That`s all! So easy and so fast.
